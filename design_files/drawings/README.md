# Technical drawings

This folder contains 2D technical drawings of the zebrafish embryo molds. These provide a quick reference for dimensions and part relationships without requiring CAD software.

## Files
- **`zfish-mold-D13_drawing.pdf`** → Drawing of the 13 mm mold (for 14 mm coverslip plates).  
- **`zfish-mold-D20_drawing.pdf`** → Drawing of the 20 mm mold (for 21 mm coverslip plates).  

## Contents
Each drawing includes:
- Overall mold dimensions (diameter, height, slot spacing).  
- Individual component views:  
  - Stamp (slot pattern and depth).  
  - Stemlock (circular locking ring).  
  - Stem (removable handle).  
- Exploded views of how components fit together.   

## Notes
- Scale in drawings: 3:1  
- Units: millimeters (mm)  
- If modifications are desired, consult the `.f3d` or `.step` files in their respective folders.
